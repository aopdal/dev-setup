# MkDocs DevContainer

This DevContainer sets up a Python 3.12 environment with MkDocs installed for documentation development.

## Usage

1. Open this folder in VS Code.
2. When prompted, reopen in container.
3. MkDocs will be available for use (`mkdocs --version`).

## Customization
- Python 3.12 base image
- Installs MkDocs via pip
- Includes Python extension for VS Code
